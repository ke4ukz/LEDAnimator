# LED Animator — device firmware (Rust)

A Rust reimplementation of the device firmware (currently the MicroPython player
generated by [`web/src/export/rp2040.ts`](../web/src/export/rp2040.ts)), targeting
the RP2040 / Pico W today and the RP2350 / Pico 2 W next. Its own crate for now;
**merges in / replaces the Python once it reaches parity.**

**Done:** the player core — WS2812 via PIO+DMA, LEDA v1 pattern parsing + fixed-rate
playback, brightness, and the LED-0 status indicator. Compiles for
`thumbv6m-none-eabi` and has been flashed and run on the bench.

**Not yet:** flash/config, BLE, Wi-Fi, multi-device sync, auth, recovery — planned
and mapped out.

👉 **Read [`docs/PORT.md`](docs/PORT.md)** for the full plan, status, the
Python→Rust mapping, build/flash/test instructions, and the toolchain gotchas.

## Quick build

```bash
export PATH="$HOME/.rustup/toolchains/stable-aarch64-apple-darwin/bin:$PATH"
cargo build --release
```
